*Percona Agent* is a background service
that collects performance data from a MySQL server.

* **Free**
  You can freely deploy Percona Agent on any Linux host with one command.
  For more information, see the :ref:`install`.

* **Open-source**
  Percona Agent source code is available on
`GitHub <https://github.com/percona/percona-agent>`_.

* **Secure**
  Percona Agent encrypts collected data and communicates with |PCT|
  through a secure websocket connection.
  For more information, see :ref:`security`.
